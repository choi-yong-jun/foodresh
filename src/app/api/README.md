# API Routes

API 라우트를 저장합니다.

## 예시 구조

```
api/
├── products/
│   └── route.ts        # GET /api/products
├── orders/
│   └── route.ts        # POST /api/orders
└── auth/
    ├── login/
    │   └── route.ts    # POST /api/auth/login
    └── register/
        └── route.ts    # POST /api/auth/register
```
